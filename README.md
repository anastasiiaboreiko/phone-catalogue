# Phone Catalog

**Phone Catalog** is a single-page web application that presents a catalog of mobile devices, allowing users to browse, explore, and compare products. 
The application is designed to provide a clear and convenient shopping-like experience, focusing on easy navigation, product discovery, and detailed product information.

The main goal of the project is to simulate a real-world e-commerce catalog, where users can filter, sort, and search products, view detailed product pages, and manage selected items. 
The application emphasizes usability, structured data presentation, and a smooth user experience across different devices.
